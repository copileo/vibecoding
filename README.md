# Hábitos — Vibecode

Aplicativo PWA de exemplo para rodar no iPhone.

## Arquivos
- `index.html`: interface e lógica
- `manifest.webmanifest`: instalação como app
- `sw.js`: funcionamento offline
- `icon.svg`: ícone

## Como testar
Publique esta pasta em um serviço de hospedagem estática, como GitHub Pages, Netlify ou Vercel.

No iPhone:
1. Abra o endereço no Safari.
2. Toque em Compartilhar.
3. Escolha "Adicionar à Tela de Início".

Os hábitos são salvos no `localStorage` do navegador.
